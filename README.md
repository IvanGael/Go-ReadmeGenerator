Readme Generator Built in Go

### Features
- Generate readme.md file based on choosed language


# Install

**Source**

1. Requirement

```
Go 1.21.4
```

2. Build binary

```
go build main.go;
```

3. Run the build

```
main
```


# Usage

Type from commandline

Example:

```
go run main.go <project_directory_path> <language>
```
